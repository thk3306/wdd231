Website Plan

Site Name: Tourism New Zealand

I chose this name to demonstrate the purpose of this Website, to encourage others to go to New Zealand and give them ideas of what kind of activites they can do.
Optional domain availability: NZtourism.co.nz

Site Purpose

The purpose of this site is to encourage people to visit New Zealand and explore the various food to eat, places to go and see.

Scenarios

What is there to see in New Zealand?
What is there to eat in New Zealand?

Color Schema

The main colors I will use will be derived from the New Zealand Flag. Blue = rgb(1, 33, 105) for the background, White = rgb(255, 255, 255) for the text and borders, and Red = rgb(200, 16, 46) for the Accents.

Fonts

I will use the Roboto font for the main body text and Inter for the headings.
